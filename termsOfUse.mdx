# 利用規約(α 版)

サービスの利用

- 本サービスは、イベントを開催し、そのチケットを電子チケットとして販売し、QR コードで受付業務を行えるサービスです。
- 開催イベントは、演奏会・講演会など、各法律と公共風俗に反しないものであれば対象を問いません。
- 本サービスはアルファ版で提供されており、本サービス上において取引された、実際のチケット売買金額とその手数料以外の如何なる損害の賠償について責任を負いません。

Stripe 利用規約への同意

- 本サービスにおける支払処理サービスは、Stripe が提供し、<a href="https://stripe.com/connect-account/legal">Stripe Connect アカウント契約</a>（<a href="https://stripe.com/legal">Stripe 利用規約</a>)を含み、総称して「Stripe サービス契約」といいます。）に従うものとします。
- 本サービスの利用の継続により、お客様は Stripe サービス契約（随時 Stripe により修正されることがあり、その場合には修正されたものを含みます。）に拘束されることに同意するものとします。
- Stripe を通じた支払処理サービスを本サービスが使用するための条件として、お客様は、本サービスに対してお客様及びお客様の事業に関する正確かつ完全な情報を提供することに同意し、このサービスが当該情報及び Stripe が提供する支払処理サービスのお客様による使用に関連する取引情報を共有することを認めるものとします。

チケットの購入・販売

- チケット購入時における決済手段として、クレジットカード・GooglePay・ApplePay に対応しています。
- サービス利用料と決済手数料として、10%が決済時に徴収されます。その費用はイベント主催者が負担します。
- 短時間の間に集中的に購入処理が行われた等で、チケットの購入から決済処理の完了までの間に在庫分チケットが無くなった場合、その注文は自動的にキャンセルされ返金されます。その際の手数料は全て本サービス運営者が負担します。

チケットの譲渡

- 本サービスは、チケットの譲渡についての機能を提供しませんが、禁止するものではありません。
- 本サービスで販売するチケットの譲渡に起因する問題について、本サービス運営者は一切の責任を負いません。

チケットの返品・返金

- チケットの返品は、基本的に行うことができません。ただし以下に該当する場合は例外とし、返金申請を行うことができます。

  - イベントの概要として記載されたものと実際の内容が極端にかけ離れていた場合
  - イベントが開催されなかった場合
  - 本サービス側の不具合や、運営の不備による損失が認められる場合

- 返金申請が行われた場合、イベント主催者は本サービス内でその通知を受け取ります。
- イベント主催者は、申請された返金申請を受け入れるかを選択できます。受け入れた場合、即時返金処理が行われます。受け入れられなかった場合、本サービス運営者による審査が行われます。
- 返金申請が 3 日間以内に処理されなかった場合、返金の申請者は本サービス上で改めて申請することで当該の返金をチケット販売者の同意なく行うことができます。
- 本サービス運営者による審査の結果は、本サービス内の通知機能で通知されます。

イベントの開催

- イベント主催者は、イベント情報を登録する前に、Stripe が要求する個人情報と本人確認用書類を提出する必要があります。
- 本人確認用書類は、提出後に Stripe による審査が行われます。
- Stripe が要求する情報が全て受け入れられた後、イベント情報の登録を行い、チケット販売を開始することができます。
- 一度登録されたイベントは削除することができません。

チケットカテゴリ

- イベントを作成した後、それに紐づくチケットカテゴリを設定できます。これは実際に販売するチケットの種類と値段、在庫数を決定するものです。
- 一度作成したチケットカテゴリは削除できません。非公開状態に変更することは可能です。

入金

- 購入されたチケット代金は通常 3 日程度の確認期間を経た後、販売者の Stripe Connect アカウントにチャージされます。
- Stripe Connect アカウントに支払い可能残高が存在する場合、月末(29-31 日)に既定の支払い口座に振り込まれます。

入会

- ユーザーはメールアドレスを登録、または各種 SNS 連携をすることによって、本サービスの会員登録、また同時に Stripe Connect アカウントが作成されることに同意したこととします。
- イベント主催者用の個人情報を登録するまで、Stripe Connect アカウントにはアカウント識別用の ID 以外の情報は記録されません。

退会

- 退会を行う際には、以下の条件を満たす必要があります。
  - ユーザーがイベントを主催している場合、イベント終了日から 10 日間以上経過している必要があります。
  - ユーザーが開催したイベントに対する返金申請が全て完了している必要があります。
  - ユーザーの振り込み待ち残高は０円になっている必要があります。入金処理をお待ちください。残高が０円にならない場合はお問い合わせください。
- 退会した場合、アカウント情報は削除され復元することはできません。ただし一部決済履歴などの情報は保管されます。
- Stripe Connect アカウント削除を希望される場合、お問い合わせからご依頼ください。

本規約の変更

- 本規約は予告なく変更される場合があります。
